This repository contains data and replication code for "Nothing to See Here? A non-inferiority approach to parallel trends" by Alyssa Bilinski and Laura Hatfield. Direct questions to alyssa_bilinski AT brown.edu.

An R package for implementing our methods can be found at: [https://github.com/laura-hatfield/NonInfParTren](https://github.com/laura-hatfield/NonInfParTren).
